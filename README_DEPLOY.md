# App Official Site

Static multilingual official website for App Store submission pages.

## Languages

- English: `/en/`
- Simplified Chinese: `/zh-Hans/`
- Russian: `/ru/`
- The root pages are English aliases for compatibility.

## Site Identity

- English brand: `Clearnova Apps`
- Chinese brand: `澄序应用`
- Support email: `support.github.io`

## Before Publishing

Replace `https://your-domain.example` in `SUBMISSION_LINKS.md`, `sitemap.xml`, and alternate links with your real HTTPS domain.

## Deploy

Upload the full `app-official-site` folder to any static host, such as GitHub Pages, Cloudflare Pages, Netlify, Vercel, S3, or another HTTPS static host.

## App Store Connect

Use `SUBMISSION_LINKS.md` as the checklist for marketing, support, and privacy URLs in each language.
