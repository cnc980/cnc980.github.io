Apple Ledger Политика конфиденциальности

Локальное финансовое приложение для книг учета, счетов, бюджетов, отчетов, регулярных записей, подписок и приватной синхронизации iCloud.

Apple Ledger Поддержка

Какие данные мы собираем

Текущая версия Apple Ledger не содержит рекламных SDK, не содержит сторонних SDK аналитики, не продает пользовательские данные и не требует учетной записи разработчика.

Финансовые данные

Apple Ledger хранит созданные вами книги учета, счета, категории, теги, продавцов, проекты, участников, операции, бюджеты, шаблоны регулярных записей, подписки, данные кредитных карт, рассрочки, курсы валют, заметки и связанные настройки.

Локальное хранение и iCloud

Финансовые данные хранятся локально на вашем устройстве. Если включить iCloud-синхронизацию, приложение синхронизирует данные через ваш приватный iCloud с помощью CloudKit.

Импорт и экспорт

Приложение может импортировать или экспортировать файлы JSON и CSV, когда вы выбираете эти действия.

Биометрическая блокировка

Apple Ledger может использовать Face ID, Touch ID или системную аутентификацию для блокировки приложения. Биометрия обрабатывается iOS, iPadOS или macOS; приложение не получает ваши биометрические данные.

Уведомления

После разрешения уведомлений приложение может использовать локальные уведомления для подписок, регулярных записей и бюджетных напоминаний.

Конфиденциальность детей

Apple Ledger не предназначено для детей и не собирает сознательно персональные данные детей.

Изменения

Если будущая версия добавит учетные записи, облачную синхронизацию разработчика, аналитику, отчеты о сбоях, платежи, рекламу или иной сбор данных, эта политика и метки конфиденциальности App Store будут обновлены.
