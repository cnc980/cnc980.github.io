SpellVPN User Guide

User Guide

Learn how to import your own VPN configuration, approve the iOS VPN permission, start a Packet Tunnel session, and manage Pro or Ultra features.

SpellVPN Support Privacy Policy
1

Before You Start

  1. Prepare a compatible sing-box JSON file, subscription URL, proxy URI, QR code, clipboard text, or local file from your own provider.
  2. Confirm that the server or subscription endpoint is reachable from your network before importing it.
  3. SpellVPN is a client. It does not sell proxy nodes or operate a developer-hosted VPN traffic network.
2

Import a Configuration

  1. Open SpellVPN and choose the import method that matches your source: QR scan, photo QR, clipboard, Files, subscription URL, proxy link, or URL scheme.
  2. Review the detected configuration name, protocol, server, and rule details before saving.
  3. If the import fails, check for an expired subscription URL, malformed JSON, unsupported protocol, or missing server and port.
3

Connect and Disconnect

  1. Select the saved configuration and tap the connect control.
  2. Approve the iOS VPN permission prompt the first time a Packet Tunnel profile is installed.
  3. Use the same control to disconnect, then review local status and logs if the tunnel does not start.
4

Backups, Logs, and Editions

  1. Use local export or iCloud Drive backup only when you want to keep a copy of your configuration data.
  2. Logs are stored on device and are useful for diagnosing connection or validation problems.
  3. SpellVPN Pro is paid, ad-free, and has no in-app purchases. SpellVPN Ultra is free with ads; optional App Store subscriptions remove ads and unlock Ultra features.
