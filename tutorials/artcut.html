ArtCut User Guide

User Guide

Create local video projects, edit clips on a timeline, add captions and overlays, adjust audio, and export finished videos.

ArtCut Support Privacy Policy
1

Create a Project

  1. Start a new project and import videos or images from Photos or Files.
  2. Grant Photos permission only when you want to import media or save exported videos.
  3. Drafts, generated renders, extracted audio, captions, and caches are stored on your device by default.
2

Edit the Timeline

  1. Arrange clips in order, then trim, split, reverse, rotate, flip, crop, add borders, or change playback speed.
  2. Use transitions, picture-in-picture frames, stickers, titles, doodles, and mosaic areas to build the final composition.
  3. Preview edits before export, especially after speed changes, reverse clips, and overlays.
3

Captions, Audio, and Export

  1. Generate automatic captions only when you explicitly start caption creation; this uses Apple's speech recognition framework.
  2. Mute, adjust volume, add fades, extract audio, or combine background sound with original clips.
  3. Choose an export quality, wait for on-device rendering to finish, then save or share through Apple system services.
4

Storage and Versions

  1. Use cache cleanup tools to remove generated renders and temporary files when storage is tight.
  2. ArtCut Free can show Yandex/AdMob ads around editing and export workflows.
  3. ArtCut Paid is a paid download with no ads. Future membership or premium template subscriptions should be checked in the submitted build before describing them as active.
